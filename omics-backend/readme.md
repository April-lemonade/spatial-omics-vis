## 🛠️ 环境准备

项目使用 [Poetry](https://python-poetry.org/) 管理 Python 依赖。

### 1. 安装 Poetry（如未安装）

```bash
curl -sSL https://install.python-poetry.org | python3 -
```
### 2. 安装依赖
```bash
poetry install
```
### 3. 启动虚拟环境
```bash
poetry shell
```

### 🚀 运行 main.py
```bash
python main.py
```
接口文档：http://localhost:8000/docs
